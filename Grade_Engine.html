TBL Grade Engine

Weighted gradebook for Team-Based Learning

Set up a course to begin.

How to use. Pick or set up a course. Add as many assessment columns as you need inside each category (labs, lecture exams, quizzes, TBLs) from + Add column. Edit category names and weights under Categories & weights. Type a raw score in each cell, up to that column's max points. Turn on Compute weighted grade to see each student's running weighted percent and letter. tRAT and any team-scored category fills every member of a student's team at once. Everything is saved in this browser only. Use Export CSV to keep your records.

Course setup

Set the course identity, weighting, grade scale, and roster. You can change all of it later.

Course identity
Options

Team-based shows a team column and lets a category fill scores across a whole team (for tRAT). Weighting can be toggled any time from the toolbar.

Categories & weights

Weights are used only when the weighted grade is on. They do not need to total 100; the running grade is computed only over categories that have at least one scored column, so partial-term grades stay fair.

Grade scale

Lower bound of each letter, as a percent. Edit to match your syllabus.

Roster

If you built this class in the Attendance Engine on this same site, pick it here to pull the enrolled roster (names, teams, confidential flags). Available only when both tools share the same web address.

CSV with a Name (or Last Name + First Name) column, optional Team and Confidential. Or one name per line as Last, First.

Categories & weights

Rename categories, set weights, and mark which ones are team-scored. Deleting a category also removes its columns.
