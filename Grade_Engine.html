TBL Grade Engine

Weighted gradebook for Team-Based Learning

Set up a course to begin.

How to use. Pick or set up a course, or build one from the Attendance Engine page you record on (upload it in Course setup). Add as many assessment columns as you need inside each category (labs, lecture exams, quizzes, TBLs) from + Add column. Edit category names and weights under Categories & weights. Type a raw score in each cell, up to that column's max points. Turn on Compute weighted grade to see each student's running weighted percent and letter.
TBL totals for Canvas. In a TBL category (the default RAT (TBL)), each column is one RAT session with separate iRAT and tRAT cells, an optional bonus, and an automatic gold Total column (iRAT + tRAT + bonus). That Total is the single number to record in Canvas. Bonus counts in the Total but is treated as extra credit in the weighted percent.
Team grades. tRAT and bonus cells are team-scored (light blue): type the score once in any member's cell and it fills the whole team. To adjust one student, for example a no-show who gets a 0, edit that filled cell; it overrides just that student (shown with a terra edge) and the rest of the team is untouched. Double-click a cell to push its value to the entire team again. Everything saves in this browser only. Use Export CSV to keep your records.

Course setup

Set the course identity, weighting, grade scale, and roster. You can change all of it later.

Course identity
Options

Team-based shows a team column and lets a category fill scores across a whole team (for tRAT). Weighting can be toggled any time from the toolbar.

Categories & weights

Weights are used only when the weighted grade is on. They do not need to total 100; the running grade is computed only over categories that have at least one scored column, so partial-term grades stay fair.

Grade scale

Lower bound of each letter, as a percent. Edit to match your syllabus.

Roster

If you built this class in the Attendance Engine on this same site, pick it here to pull the enrolled roster (names, teams, confidential flags). Available only when both tools share the same web address.

Upload the Attendance Engine HTML page you record on and the roster (names, teams, confidential flags) is read straight from it. A CSV (Name, optional Team, Confidential) or a saved attendance CSV also work.

To export from the Attendance Engine: open your class, then in your browser choose File then Save Page As and pick “Webpage, HTML Only”. Upload that file here. Names and confidential flags import exactly; team names import when your browser saves the selected team.

Categories & weights

Rename categories, set weights, and mark which ones are team-scored. Deleting a category also removes its columns.

Add column

A column is one graded item: a lab, a lecture exam, a quiz, or (in a TBL category) one RAT session that totals iRAT + tRAT.

Column

This adds one RAT session with separate iRAT and tRAT cells (tRAT fills the team) and an automatic Total column (iRAT + tRAT + bonus) for you to record in Canvas. Bonus is extra credit and is not counted in the category percent.
